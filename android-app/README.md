# android-app

Minimal Android application shell for the Anki MCQ Generator project.

## What's included
- Kotlin-based Android app module (`app`)
- A single `MainActivity`
- Basic launcher resources and app theme

## Next steps
1. Add Gradle wrapper files (`gradlew`, `gradlew.bat`, and `gradle/wrapper/*`) from Android Studio or `gradle wrapper`.
2. Open `android-app/` in Android Studio.
3. Sync and run on an emulator/device.
